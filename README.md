Part 1 focuses on project planning, research, content development, and website structure. This includes defining goals and objectives, analysing the target audience, planning the website structure, and creating initial wireframes.

 Website Goals
- To introduce Sparkle Cleaning Solutions to potential customers
- To clearly communicate available cleaning services
- To provide easy navigation and access to contact information
- To create a professional and trustworthy brand image

Target Audience
The target audience includes homeowners, small businesses, and corporate clients looking for reliable and affordable cleaning services.

- About Us
- Services
- Enquiry
- Contact

- Part 2 builds on the foundation created in Part 1 by focusing on visual design, layout, and responsive styling. An external CSS stylesheet was created and applied across all pages to ensure consistency and professional appearance.

The website is styled for desktop devices and enhanced with responsive design techniques to ensure usability across tablets and mobile devices.

Key Features Implemented
- External CSS stylesheet linked to all pages
- Consistent colour scheme and typography
- Flexbox and Grid layouts for structured content
- Pseudo-classes for interactive elements
- Responsive design using media queries

Responsive Design
The website layout adapts to different screen sizes by adjusting:
- Navigation layout
- Content columns
- Font sizes and spacing
